Huffman Kodierung

Diese Demonstration soll die Funktionsweise des Huffman Kodierung Plug-ins zeigen.

Das Plug-in Huffman Kodierung lässt sich über das Menü Visualisierungen oder über den Krypto-Explorer im Tab Visualisierungen starten und in seiner Ansicht maximieren.

Die Sicht des Plug-ins besteht aus drei Registerkarten: Parameter, Huffmanbaum und Codetabelle.

Auf der ersten Registerkarte Parameter hat man zwei Szenarien, aus denen man wählen kann:

Szenario 1 - Komprimieren:
In diesem Szenario geht es um die Komprimierung von Text. Dabei hat man die Möglichkeit den Text aus einer Datei zu laden oder ihn manuell in das Feld Eingabetext einzugeben. Für das Laden einer Datei kann man folgendermaßen vorgehen: Per Klick auf den Button Unkomprimierte Datei öffnen... wird der Dateiauswahl-Dialog des Betriebssystems geöffnet und nach Textdateien (ISO-8859-1) gefiltert.

Nachdem man die Datei ausgewählt hat, wird der Inhalt der Datei in das Feld Eingabetext geladen und der Button Komprimieren wird aktiviert. Die Dateigröße wird ermittelt und neben dem Text Dateigröße (in Bytes) angezeigt. Dadurch kann man nach der Komprimierung die Dateigrößen vergleichen.

Für das manuelle Eingeben eines Freitextes kann man direkt im Feld Eingabetext einen gewünschten Freitext eingeben. Sobald der Freitext eingegeben wurde, wird der Button Unkomprimierte Datei öffnen... deaktiviert. Somit kann man zwischen manueller Texteingabe und Laden einer Datei wählen bzw. zum Initialen-Zustand zurückkehren, in dem man den zuvor manuell eingegeben Text wieder löscht. Dadurch wird der Unkomprimierte Datei öffnen... Button wieder aktiviert.

Durch Klick auf den Komprimieren Button wird der Text im Feld Eingabetext komprimiert und eine Ausgabedatei mit der Dateierweiterung .huffman wird im Home-Verzeichnis des Benutzers erzeugt. Der Dateipfad und die Größe der Ausgabedatei werden angezeigt. Der Button Komprimieren wird deaktiviert und der Button Dekomprimieren aktiviert.

Nun kann man durch Klick auf den Button Dekomprimieren der zuvor komprimierte Text dekomprimieren. Dabei wird der dekomprimierte Text im Feld Ausgabetext angezeigt.

Szenario 2 - Dekomprimieren
In diesem Szenario kann man eine komprimierte Datei dekomprimieren. Dafür klickt man auf den Button Komprimierte Datei öffnen..., um den Dateiauswahl-Dialog des Betriebssystems zu öffnen. Dabei wird nach .huffman-Dateien gefiltert.

Nach dem eine .huffman-Datei ausgewählt wurde, wird die Dateigröße ermittelt und neben dem Text Dateigröße (in Bytes) angezeigt und der Button Dekomprimieren ist aktiviert.

Durch Klick auf den Button Dekomprimieren wird die Datei dekomprimiert und sein Inhalt in dem Feld Ausgabetext angezeigt.

Es ist jederzeit möglich durch Klicken auf dem Button Eingaben zurücksetzen auf den Initialen-Zustand zurückzukehren und den Algorithmus von vorne zu beginnen.

Je nachdem ob eine Komprimierung bzw. Dekomprimierung durchgeführt wird, wird auf der zweite Registerkarte, Huffmanbaum die Baumstruktur dargestellt. Das Huffmanbaum ist ein Binärbaum, der an seinen Blättern die Informationen trägt. Die Blätter sind Grün markiert und stellen die ASCII-Zeichen des Eingabetextes dar. Auf den Kanten des Graph werden die Informationen 1 oder 0 dargestellt, die für die Zusammenstellung der Codewörter für das jeweilige Zeichens verwendet wird. Beim Wurzelknoten und bei den inneren Knoten eines Baums, der durch Komprimierung erstellt wurden, werden die Wahrscheinlichkeiten der Zeichen dargestellt. Bei einem Baum, der durch Dekomprimieren erzeugt wurde, wird der Wurzelknoten bzw. die inneren Knoten als leere Knoten dargestellt, da in diesem Fall keine Wahrscheinlichkeiten berechnet werden.

Wenn man mit der Maus über ein Blatt geht, wird als Tooltip das Codewort des Zeichens angezeigt.

Beim Klick auf einer der Blätter wird der dazugehörigen Zweig selektiert. Der gesamte Zweig von der Wurzel bis zum angeklickten Blatt wird farblich hervorgehoben und man kann den Zweig bei Bedarf verschieben. Das Codewort des augewählten Blattes wird oben angezeigt.

Auf der dritten Registerkarte wird die Codetabelle angezeigt. Dabei wird für jedes ASCII-Zeichen, die in dem Eingabetext vorkommt, das dazu gehörige Codewort angezeigt. Beim Klick auf den Button Zweig anzeigen, wechselt die Ansicht auf die Huffmanbaum Registerkarte und selektiert den dazupassenden Zweig.

In der Menüleiste kann man durch Klicken auf das -Icon analog zum Eingaben zurücksetzen Button auf den Initialen-Zustand zurückkehren und den Algorithmus von vorne zu beginnen.

Man kann durch Klicken auf das -Icon in der Menüleiste das Layout des Huffmanbaums dynamisch ändern. Standardmäßig wird der Huffmanbaum in Baum Layout angezeigt. Zusätzlich sind die Horizontal Baum Layout und Radial Layout verfügbar. Unterschiedliche Darstellungsmöglichkeiten des Huffmanbaums sollen die Lesbarkeit des Baums vereinfachen.

Die Größe des Huffmanbaums wird beim Erstellen direkt an die Größe der View angepasst, so dass er vollständig angezeigt wird. Die Ansicht kann aber auch per Zoom-Funktion über das -Icon skaliert werden.
